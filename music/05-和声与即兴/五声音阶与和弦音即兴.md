# 🎵 五声音阶与和弦音即兴

> 最简单的即兴方法，用五声音阶和和弦音创造旋律

---

## 五声音阶

### 什么是五声音阶
五声音阶由五个音组成，没有半音，听起来和谐。

### C 大调五声音阶
```
C - D - E - G - A - C
1 - 2 - 3 - 5 - 6 - 1
```

### A 小调五声音阶
```
A - C - D - E - G - A
1 - b3 - 4 - 5 - b7 - 1
```

### 练习要点
- 没有半音，所以怎么弹都好听
- 适合初学者即兴
- 可以在任何调上使用

---

## 和弦音即兴

### 什么是和弦音即兴
只用和弦内的音即兴，保证音符与和弦和谐。

### C 和弦的和弦音
```
C - E - G
根音 - 三度 - 五度
```

### 练习方法
1. 弹一个 C 和弦
2. 只用 C、E、G 三个音即兴
3. 尝试创造简单的旋律

---

## 🎯 练习建议

### 每日练习
1. 选择一个调的五声音阶
2. 用五声音阶即兴 5 分钟
3. 用和弦音即兴 5 分钟

### 练习顺序
1. C 大调五声音阶
2. C 和弦的和弦音
3. G 大调五声音阶
4. G 和弦的和弦音

---

## 🎵 即兴技巧

### 技巧一：重复
- 重复一个音型
- 创造记忆点

### 技巧二：变化
- 改变节奏
- 改变音高

### 技巧三：留白
- 不要一直弹
- 留出空间
