# 🎵 三和弦

> 三和弦是最基础的和弦，由三个音组成

---

## 什么是三和弦

三和弦由三个音组成：
- **根音**：和弦的基础音
- **三度**：决定和弦的性质（大三/小三）
- **五度**：和弦的稳定音

---

## 大三和弦

### 构成
- 根音 + 大三度 + 纯五度
- 例如：C = C + E + G

### 常见大三和弦
| 和弦 | 组成音 | 键盘位置 |
|------|--------|----------|
| C | C-E-G | C + E + G |
| D | D-F#-A | D + F# + A |
| E | E-G#-B | E + G# + B |
| F | F-A-C | F + A + C |
| G | G-B-D | G + B + D |
| A | A-C#-E | A + C# + E |
| B | B-D#-F# | B + D# + F# |

### 练习要点
- 三个音要同时按下
- 手指站稳
- 听和弦的共鸣

---

## 小三和弦

### 构成
- 根音 + 小三度 + 纯五度
- 例如：Cm = C + Eb + G

### 常见小三和弦
| 和弦 | 组成音 | 键盘位置 |
|------|--------|----------|
| Cm | C-Eb-G | C + Eb + G |
| Dm | D-F-A | D + F + A |
| Em | E-G-B | E + G + B |
| Fm | F-Ab-C | F + Ab + C |
| Gm | G-Bb-D | G + Bb + D |
| Am | A-C-E | A + C + E |
| Bm | B-D-F# | B + D + F# |

### 练习要点
- 与大三和弦的区别是三度
- 小三和弦听起来更忧郁

---

## 增三和弦

### 构成
- 根音 + 大三度 + 增五度
- 例如：Caug = C + E + G#

### 特点
- 听起来紧张、不稳定
- 较少使用

---

## 减三和弦

### 构成
- 根音 + 小三度 + 减五度
- 例如：Cdim = C + Eb + Gb

### 特点
- 听起来紧张、不安
- 常用于过渡

---

## 🎯 练习建议

### 每日练习
1. 选择一个调的和弦
2. 左手弹根音，右手弹和弦
3. 练习和弦转换

### 练习顺序
1. C 大调的 I-IV-V-I
2. G 大调的 I-IV-V-I
3. F 大调的 I-IV-V-I
4. D 大调的 I-IV-V-I
