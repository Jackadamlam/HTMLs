# 🎵 和弦转位

> 和弦转位可以让手移动更少，弹奏更流畅

---

## 什么是和弦转位

和弦转位是指改变和弦中音的顺序：
- **原位**：根音在最低
- **第一转位**：三度在最低
- **第二转位**：五度在最低

---

## C 和弦的转位

### 原位
```
C - E - G
1 - 3 - 5
```

### 第一转位
```
E - G - C
3 - 5 - 1
```

### 第二转位
```
G - C - E
5 - 1 - 3
```

---

## G 和弦的转位

### 原位
```
G - B - D
1 - 3 - 5
```

### 第一转位
```
B - D - G
3 - 5 - 1
```

### 第二转位
```
D - G - B
5 - 1 - 3
```

---

## F 和弦的转位

### 原位
```
F - A - C
1 - 3 - 5
```

### 第一转位
```
A - C - F
3 - 5 - 1
```

### 第二转位
```
C - F - A
5 - 1 - 3
```

---

## 为什么用转位

### 优点
1. **手移动更少**：相邻和弦可以用最近的转位
2. **声音更流畅**：低音线条更平滑
3. **更容易弹奏**：减少大跳

### 例子
从 C 到 G：
- 原位：C-E-G → G-B-D（手要跳）
- 转位：C-E-G → B-D-G（手只需移动一点）

---

## 🎯 练习建议

### 练习方法
1. 先练习原位和弦
2. 再练习转位
3. 练习和弦转换时用转位

### 练习顺序
1. C 和弦的三个位置
2. G 和弦的三个位置
3. F 和弦的三个位置
4. 练习 C → G → F → C 的转位连接
