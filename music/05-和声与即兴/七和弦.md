# 🎵 七和弦

> 七和弦在三和弦基础上增加一个音，让和声更丰富

---

## 什么是七和弦

七和弦由四个音组成：
- **根音**：和弦的基础音
- **三度**：决定和弦的性质
- **五度**：和弦的稳定音
- **七度**：增加色彩

---

## 大七和弦（Maj7）

### 构成
- 根音 + 大三度 + 纯五度 + 大七度
- 例如：Cmaj7 = C + E + G + B

### 常见大七和弦
| 和弦 | 组成音 | 键盘位置 |
|------|--------|----------|
| Cmaj7 | C-E-G-B | C + E + G + B |
| Fmaj7 | F-A-C-E | F + A + C + E |
| Gmaj7 | G-B-D-F# | G + B + D + F# |

### 特点
- 听起来温暖、柔和
- 常用于抒情歌

---

## 属七和弦（7）

### 构成
- 根音 + 大三度 + 纯五度 + 小七度
- 例如：C7 = C + E + G + Bb

### 常见属七和弦
| 和弦 | 组成音 | 键盘位置 |
|------|--------|----------|
| C7 | C-E-G-Bb | C + E + G + Bb |
| D7 | D-F#-A-C | D + F# + A + C |
| E7 | E-G#-B-D | E + G# + B + D |
| G7 | G-B-D-F | G + B + D + F |

### 特点
- 听起来有张力
- 常用于 blues 和 jazz
- 解决到主和弦

---

## 小七和弦（m7）

### 构成
- 根音 + 小三度 + 纯五度 + 小七度
- 例如：Cm7 = C + Eb + G + Bb

### 常见小七和弦
| 和弦 | 组成音 | 键盘位置 |
|------|--------|----------|
| Cm7 | C-Eb-G-Bb | C + Eb + G + Bb |
| Dm7 | D-F-A-C | D + F + A + C |
| Em7 | E-G-B-D | E + G + B + D |
| Am7 | A-C-E-G | A + C + E + G |

### 特点
- 听起来柔和、忧郁
- 常用于流行和 R&B

---

## 减七和弦（dim7）

### 构成
- 根音 + 小三度 + 减五度 + 减七度
- 例如：Cdim7 = C + Eb + Gb + Bbb

### 特点
- 听起来紧张、不安
- 较少使用

---

## 🎯 练习建议

### 练习方法
1. 先掌握三和弦
2. 再学习七和弦
3. 练习和弦转换

### 练习顺序
1. 属七和弦（G7、C7、D7）
2. 小七和弦（Am7、Dm7、Em7）
3. 大七和弦（Cmaj7、Fmaj7）
