# 👂 听大/小和弦

> 听出和弦是大三和弦还是小三和弦，是听力训练的重要一步

---

## 大三和弦 vs 小三和弦

### 大三和弦
- **构成**：根音 + 大三度 + 纯五度
- **感觉**：明亮、快乐、稳定
- **例子**：C = C + E + G

### 小三和弦
- **构成**：根音 + 小三度 + 纯五度
- **感觉**：忧郁、柔和、不稳定
- **例子**：Cm = C + Eb + G

---

## 训练方法

### 方法一：对比听辨
1. 弹 C 和弦（大三）
2. 弹 Cm 和弦（小三）
3. 感受两者的区别

### 方法二：歌曲中听辨
1. 听一首歌
2. 尝试判断和弦是大三还是小三
3. 在键盘上验证

### 方法三：和弦进行听辨
1. 弹一个和弦进行
2. 尝试判断每个和弦的性质
3. 在键盘上验证

---

## 🎯 练习建议

### 每日练习
1. 弹 C 和弦，感受大三的感觉
2. 弹 Cm 和弦，感受小三的感觉
3. 随机弹一个和弦，判断是大三还是小三

### 练习顺序
1. C 和 Cm 的对比
2. G 和 Gm 的对比
3. F 和 Fm 的对比
4. 随机和弦判断

---
