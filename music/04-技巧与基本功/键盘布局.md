# 🎹 键盘布局

> 认识键盘，找到所有音

---

## 键盘结构

### 白键
- C - D - E - F - G - A - B
- 7 个音为一组，循环重复

### 黑键
- 升号（#）：升高半音
- 降号（b）：降低半音
- 例如：C# = Db，D# = Eb

---

## 找中央 C

### 方法一：数白键
1. 找到键盘最左边的 C
2. 往右数 4 个 C
3. 就是中央 C

### 方法二：数黑键
1. 找到两个黑键的组合
2. 两个黑键左边的白键是 C
3. 往右数 4 个 C

---

## 键盘图示

```
C  D  E  F  G  A  B  C
|  |  |  |  |  |  |  |
```

### 黑键位置
- C# / Db：C 和 D 之间
- D# / Eb：D 和 E 之间
- F# / Gb：F 和 G 之间
- G# / Ab：G 和 A 之间
- A# / Bb：A 和 B 之间
