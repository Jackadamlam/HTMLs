# 🥁 节奏型库

> 不同的节奏型可以让歌曲更有表现力

---

## 基础节奏型

### 四分音符节奏
```
| X - X - | X - X - |
| 1 - 2 - | 3 - 4 - |
```
- **适用**：简单歌曲、儿歌
- **练习**：每拍弹一次和弦

### 八分音符节奏
```
| X X X X | X X X X |
| 1 + 2 + | 3 + 4 + |
```
- **适用**：流行歌、抒情歌
- **练习**：每拍弹两次和弦

### 十六分音符节奏
```
| X X X X X X X X | X X X X X X X X |
| 1 e + a 2 e + a | 3 e + a 4 e + a |
```
- **适用**：快歌、摇滚
- **练习**：每拍弹四次和弦

---

## 分解和弦节奏型

### 基础分解
```
| 1 - 3 - | 5 - 3 - |
```
- **适用**：抒情歌、 ballad
- **练习**：根音-三度-五度-三度

### 流行分解
```
| 1 3 5 3 | 1 3 5 3 |
```
- **适用**：流行歌
- **练习**：根音-三度-五度-三度

### 琶音分解
```
| 1 - 3 5 | 3 - 5 1 |
```
- **适用**：抒情歌
- **练习**：根音-三度-五度-三度-五度-根音

---

## 柱式和弦节奏型

### 基础柱式
```
| X - X - | X - X - |
```
- **适用**：进行曲、庄严歌曲
- **练习**：每拍弹一次完整和弦

### 切分柱式
```
| X - X X | X - X X |
| 1 - 2 + | 3 - 4 + |
```
- **适用**：流行歌、R&B
- **练习**：有切分的感觉

---

## 半分解和弦节奏型

### 基础半分解
```
| 1 - 3 5 | 1 - 3 5 |
```
- **适用**：流行歌
- **练习**：根音-和弦

### 流行半分解
```
| 1 35 35 | 1 35 35 |
```
- **适用**：流行歌
- **练习**：根音-和弦-和弦

---

## 🎯 练习建议

### 练习方法
1. 先用单个和弦练习节奏型
2. 再用和弦进行练习
3. 最后用歌曲练习

### 练习顺序
1. 四分音符节奏
2. 八分音符节奏
3. 分解和弦
4. 半分解和弦
